## Crypto Project

A crypto tracker to view latest data. \
Also a platform to practice trading crypto with virtual currency.

## Tools

- Frontend
  - ReactJS
  - Chakra UI
  - Tailwind CSS
  - Redux Toolkit
  - Firebase (For auth)
  - Typescript
- Backend
  - NodeJS
  - Express
  - PostgreSQL
  - Typescript
